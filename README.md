# Explainable AI Loan Prediction

## Project Overview
This project predicts loan approval and explains model decisions using SHAP.

## Dataset
Loan prediction dataset with borrower financial and credit information.

## Project Workflow
1. Data Exploration
2. Data Preprocessing
3. Model Training
4. Model Evaluation
5. Explainable AI using SHAP

## Model
Logistic Regression

## Explainability
SHAP is used to interpret feature importance and individual predictions.
